우타우 자동 원음설정기능을 만들어볼 수 있을까?
모리샘플러는 화칸루가 개발한 우타우용 음성합성 엔진인데 그 안에 자동으로 원음설정을 해주는 기능이 있어. 알파싱, 일본어 CVVC, VCV 가 있고 한국어를 지원하지 않아서 아쉬웠어.
원리는 비교적 간단해. _ga'gi'gu'ge'go 라는 파일이 있고, oto.ini 내부에 _ga'gi'gu'ge'go 라는 오토들이 있을꺼야. 연속음이라면?
_ga'gi'gu'ge'go.wav=- ga,0,0,0,0,0
_ga'gi'gu'ge'go.wav=a gi,0,0,0,0,0
_ga'gi'gu'ge'go.wav=i gu,0,0,0,0,0
_ga'gi'gu'ge'go.wav=u ge,0,0,0,0,0
_ga'gi'gu'ge'go.wav=e go,0,0,0,0,0
와 같아. 이 발음기호에 따라 자동으로 원음설정을 해줘.
파형을 분석 > 다음과 같이 형식에 따라 자동 원음설정.

단독음 [CV], [- CV], [V], [- V] :
오프셋 : 녹음한 파일의 시작 ( "가" 를 녹음한다 치면 "가" 녹음을 시작하는 순간. ) 
오버렙 : 선행과 같은 위치
선행 : 자음과 모음의 사이 / 경계선 ( 모음은 오프셋과 비슷. 허나 조금 앞에. )
고정 : 모음이 안정되기 시작하는 부분. ( 서스테인의 시작.)
컷오프 : 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)

연속음 [V CV] 발음기호 :
오프셋 : 이전 발음 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)
오버렙 : 선행과 오프셋 사이
선행 : 자음과 모음의 사이 / 경계선
고정 : 모음이 안정되기 시작하는 부분. ( 서스테인의 시작.)
컷오프 : 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)

연속음 [V V] 발음기호 :
오프셋 : 이전 발음 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)
오버렙 : 선행과 오프셋 사이
선행 : 오프셋, 컷오프 기준, 3/1 지점
고정 : 모음이 안정되기 시작하는 부분. ( 서스테인의 시작.)
컷오프 : 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)

CVVC 에 [V C] 발음 기호
오프셋 : 이전 발음 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)
오버렙 : 선행과 오프셋 사이
선행 : 자음과 모음의 사이 / 경계선
고정 : 선행, 컷오프 사이
컷오프 : 다음 발음이 시작하는 부분.

어미숨 [V R] : 
오프셋 : 이전 발음 모음이 안정되는 부분이 끝나는 부분. ( 서스테인의 끝.)
오버렙 : 선행과 오프셋 사이
선행 : 모음과 한숨소리 경계선
고정 : 한숨 소리가 끝나는 지점
컷오프 : 고정보다 조금 앞에.

CV 에는 단독음 발음기호를 원음설정.
VCV 에는 단독음, 연속음 발음기호를 원음설정.
CVVC 에는 단독음과 CVVC 발음기호로 원음설정.

기호는 "처음" 을 의미함. "안녕 [an nyeong]" 을 CVVC 기준 우타우에서 쓴다면
a / a n / nyeo / eo ng 이 아닌,
"- a" / a n / nyeo / eo ng 로 쓸 수 있는 기호임.

R 은 대중적으로 한숨임.
감정을 담는 소리, 우울한 분위기, 끝맺을때 넣어서 내는 소리임. "안돼 [an dwe]" 를 CVVC 기준 우타우에서 쓴다면
a / a n / dwe 가 아닌,
a / a n / dwe / e R 로 쓸 수 있는거임.
