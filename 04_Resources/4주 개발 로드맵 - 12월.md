## 1. 트랙 정의

- **트랙 V (Vegas)**  
  VEGAS Pro 음성 인식 확장 POC  
  - 선택 구간 → 오디오 추출 → ASR → LLM 후처리 → 타임라인 자막/마커 생성

- **트랙 K (ko-G2P / aligner)**  
  한국어 G2P + forced aligner 기반 프론트엔드 파이프라인  
  - GTSinger 데이터셋 전처리  
  - 규칙 기반 ko-G2P  
  - 기존 aligner 래핑 → E2E alignment 데모

- **트랙 G (gradio)**  
  `gradio-pianoroll` 유지보수 및 릴리즈  
  - 코드 리팩터링 / 테스트 / README / 예제 / PyPI 릴리즈

- **트랙 A (git MCP)**  
  git analysis MCP 서버 개발  
  - git 리포 구조 분석, 최근 변경 요약, hotspot 파일 분석 등  
  - LLM이 코드베이스를 이해하는 데 쓰는 MCP 툴

- **공통 트랙 M (Math/DSP)**  
  매일 30–60분: DSP / 수학 / 통계 / 최적화 공부  
  - 그날 메인 트랙과 직접 관련된 내용 위주로 학습
