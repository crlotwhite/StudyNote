# Proj-vits-visinger-styletss-roadmap

## 0. 개요
- 목표: VITS → VISinger → StyleTTS 논문 구현/재현을 위해, 선행 DSP + 핵심 딥러닝 구조(Flow/MAS 등)를 먼저 학습한 뒤 단계적으로 구현 역량을 확보
- 마감: Day1~18 학습/구현 로드맵 완주(18일 플랜, 시작일은 별도 지정)

## 1. 로드맵
- [ ] (Day1~?) DSP 기초 정리: 멜 스펙트로그램, STFT/ISTFT, 로그-스케일, 윈도잉/홉, 프레임 개념
- [ ] (Day1~?) F0/소스-필터 개요: F0 추정, 보이싱/언보이싱, 소스-필터 모델, 스펙트럼 엔벨로프 관점
- [ ] (Day?~?) Flow 기본기: Normalizing Flow 개념, likelihood 관점, invertible transform, 학습 안정화 포인트
- [ ] (Day?~?) MAS( monotonic alignment search ) 이해: 강제 정렬 관점, 길이/어텐션 정렬 이슈와 해결 방식
- [ ] (Day?~?) VITS 구현 단계:
  - [ ] 데이터 파이프라인(텍스트/발음/정렬 입력 형태) 구성
  - [ ] 핵심 모듈(인코더/디코더/flow/align) 구현 및 학습 루프 연결
  - [ ] 최소 재현(음질/정렬 동작 확인) 및 문제 원인 분해(데이터/정렬/보코더/학습 안정성)
- [ ] (Day?~?) VISinger 확장 단계:
  - [ ] VITS 기반에서 SVS에 필요한 조건(음소+노트/타이밍 등) 입력 설계
  - [ ] 노래 발성(피치/길이/강세) 조건 처리 및 정렬/길이 예측 체계 점검
- [ ] (Day?~?) StyleTTS 확장 단계:
  - [ ] 스타일 레이어(스타일 임베딩/컨디셔닝) 구조 이해 및 적용 지점 정의
  - [ ] 스타일 제어/일관성/전이(transfer) 실험 플랜 수립
- [ ] (중/후반) PSOLA 심화 실습:
  - [ ] TD-PSOLA 기본 구현/실험(피치/타임 스케일 변환)
  - [ ] 딥러닝 파이프라인에서의 역할(보정/후처리/데이터 증강 등) 아이디어 정리
